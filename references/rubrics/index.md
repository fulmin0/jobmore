# Rubrics Index

All rubrics grounded in *Cracking the PM Interview* (McDowell & Bavaro, 2014).
Each rubric scores 1–5 per dimension. A score of 1 in any dimension caps the overall at 3.0.

| File | Artifact | Perspective | When to Run |
|------|----------|-------------|-------------|
| [elevator_pitch.md](elevator_pitch.md) | LinkedIn DM / InMail to HM | HM receiving 20 cold DMs/week | After job_intelligence, before resume content |
| [resume.md](resume.md) | One-page tailored resume | HM reading cold, 15-second skim | After resume content is generated, before PDF build |
| [cover_letter.md](cover_letter.md) | LaTeX cover letter, 3 paragraphs | HM who has already seen the resume | After resume content is approved |

## Shared Rules Across All Three

- **Cold-read first**: Every rubric starts with a cold-read pass before scoring. Write down the unfiltered first impression — it informs every dimension score.
- **1-in-any-dimension cap**: A score of 1 on any dimension caps the total at 3.0, regardless of other scores. A fatal gap in one area cannot be hidden by strong performance elsewhere.
- **Specificity over impressiveness**: The most common failure across all three artifacts is generic content that sounds good but could apply to any company or any candidate. Specificity — named products, named decisions, named outcomes — is what separates 3 from 4.
- **Additionality**: Each artifact in the pipeline should add something the previous one didn't. The elevator pitch is a hook. The resume is evidence. The cover letter is context and narrative. If they all say the same things, the candidate is wasting the reader's attention.
